## Coverage Report

The detailed HTML coverage report is generated by the [GitHub Actions workflow](https://github.com/dertnius/kafka-producer-balanced/actions/workflows/test-and-coverage.yml) on every push.

Once the workflow completes successfully, the full interactive coverage report will be available at:
- **Location**: `docs/coverage/` directory
- **View**: Check GitHub Actions runs for detailed logs

## Running Tests Locally

To generate and view the coverage report on your machine:

```powershell
cd MyDotNetSolution
dotnet test Tests/MyDotNetApp.Tests/MyDotNetApp.Tests.csproj `
  --configuration Release `
  --collect:"XPlat Code Coverage" `
  --results-directory:TestResults
```
